Para lograr una conversación efectiva es importante entender el contexto donde se está desarrollando. En esta sección aprenderás a usar las cartas a partir de simulaciones, realizando un recorrido donde deberás definir:

  • El entorno
  • La situacion
  • Como te sientes?
  • Que sientes?
  • Que pregunta te harias?
  • Distincion de conceptos
  • Recursos para usar

En cada una de ellas deberás hacer una elección que te permitirá armar una combinación de cartas y, al finalizar, te daremos algunas recomendaciones para lograr una conversación inteligente, según las decisiones que hayas tomado.
